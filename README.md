# countrycodedetector
A simple web app that detects the country of origin from an international phone number using dial code prefix matching.
A simple HTML/CSS/JS app that identifies the country of origin of a phone number based on its international dial code prefix.

## How it works
Enter a phone number starting with `+` and the app checks the prefix against known country dial codes using if/else statements.

## Features
- Detects 3-digit codes (e.g. +234 → Nigeria)
- Detects 2-digit codes (e.g. +44 → United Kingdom)
- Detects 1-digit codes (e.g. +1 → United States / Canada)
- Live result as you type

## Tech Stack
